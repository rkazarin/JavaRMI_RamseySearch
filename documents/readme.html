How to use program

**To Compile:**

cd JavaRMI_RamseySearch
ant dist

**To Execute Space:**

ant -Darg0={numLocalThreads} runSpace

*To Execute Computer (in seperate terminal window:)**

ant -Darg0={remote_space_host_ip_address} -Darg1={desiredPrefetchBufferSize} -Darg2={desiredNumThreads} -Darg3={isLongRunning} runComputer

*To Execute Computer Graph Store for Ramsey:**

ant runRamseyStore

**To Execute TSP Client (in seperate terminal window:):**

ant -Darg0={remote_space_host_ip_address} -Darg1={12, 13, 16 cities} -Darg2={true or false branchAndBound} runTspClient

**To Execute Ramsey Client (in seperate terminal window:):**

ant -Darg0={remote_space_host_ip_address} -Darg1={graph_store_host_ip_address} runRamseyClient
